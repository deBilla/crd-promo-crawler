# Architecture Diagrams

This directory contains Excalidraw source files for the project's architecture diagrams.

## How to Edit

1. Open [excalidraw.com](https://excalidraw.com)
2. Use **Open** to load a `.excalidraw` file from this directory
3. Edit the diagram
4. **Save to disk** back to this directory
5. Commit the updated file

Alternatively, use the [Excalidraw VS Code extension](https://marketplace.visualstudio.com/items?itemName=pomdtr.excalidraw-editor) to edit directly in your IDE.

## Files

| File | Description |
|------|-------------|
| `architecture.excalidraw` | Full system topology — services, queues, infrastructure, data flow |

## Conventions

- **Black (`#1e1e1e`)** — Existing/original elements
- **Green (`#2f9e44`)** — New or changed elements (dashed arrows for new connections)
- **Rectangles** — Services and queues
- **Ellipses** — Databases and storage
- **Arrows** — Data flow connections
